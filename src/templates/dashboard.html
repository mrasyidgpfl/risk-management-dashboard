CLI Endpoints

Read-only point-in-time views into engine state, for when the WebSocket throttle (~250ms) is too slow. JSON by default; append ?format=table for ASCII output.

EndpointDescription
GET /api/summaryTop-line PnL, trade count, instrument count
GET /api/ticksLatest tick for every instrument
GET /api/ticks/<instrument>Latest tick for one instrument (e.g. EURUSD)
GET /api/positionsAll positions with PnL breakdown
GET /api/positions/<instrument>One position drill-down
GET /api/clientsClient metrics, sorted by volume desc
Examples
curl -s localhost:8080/api/summary | jq
                curl -s 'localhost:8080/api/ticks/EURUSD'
                curl -s 'localhost:8080/api/positions?format=table'
                curl -s 'localhost:8080/api/clients?format=table'
